# Flower Shop Website

A modern and responsive Flower Shop web application built using HTML, CSS, JavaScript, and Bootstrap. 
The project provides a clean shopping experience with user authentication simulation,
responsive navigation, and multiple pages for browsing flowers and managing orders.

---

## 🚀 Features

- Responsive design for desktop, tablet, and mobile
- Modern and clean user interface
- Responsive navigation bar with mobile menu
- User Login & Sign Up interface
- LocalStorage-based authentication simulation
- Dynamic navigation based on user login status
- About page
- Contact page with contact form
- Orders page
- Smooth animations and hover effects

---


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- LocalStorage

---
## 🎯 Project Objectives

- Build a responsive e-commerce website.
- Practice responsive web design principles.
- Improve JavaScript DOM manipulation skills.
- Simulate user authentication using LocalStorage.
- Develop a clean and reusable UI.
